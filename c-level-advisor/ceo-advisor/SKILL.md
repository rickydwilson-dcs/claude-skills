# CEO Advisor

Executive leadership guidance for strategic decision-making, organizational development, and stakeholder management.

## Overview

The CEO Advisor skill provides comprehensive frameworks, tools, and templates for chief executive leadership. It combines strategic planning methodologies, financial modeling, stakeholder management frameworks, and executive communication templates to support CEOs in driving organizational success.

This skill supports the full spectrum of CEO responsibilities: setting vision and strategy, managing capital and resources, leading stakeholders, building organizational culture, and representing the company externally. All content is designed for immediate application in board meetings, investor relations, strategic planning sessions, and day-to-day executive decision-making.

## Core Capabilities

**Strategic Planning & Analysis**
- SWOT, Porter's Five Forces, BCG Matrix analysis
- Scenario planning and option evaluation
- Market positioning and competitive strategy
- Strategic initiative prioritization
- Python-based strategy analyzer tool

**Financial Leadership**
- Capital allocation frameworks and priorities
- Multi-scenario financial modeling with NPV/IRR
- Fundraising strategy (Seed through IPO)
- Budget planning and resource allocation
- Python-based financial scenario analyzer

**Board & Investor Relations**
- Board meeting preparation and packages
- Investor communication templates
- Quarterly and annual reporting
- Crisis communication frameworks
- Governance best practices

**Organizational Leadership**
- Culture development and transformation
- Executive routines and time management
- Performance management and metrics
- Succession planning frameworks
- Personal development strategies

**Decision-Making Frameworks**
- Go/no-go decision checklists
- Risk assessment and mitigation
- Stakeholder priority matrices
- Crisis management playbooks
- Strategic trade-off analysis

## Quick Start

### Strategic Analysis

```bash
# Analyze strategic position using SWOT, Porter's Five Forces, BCG Matrix
python scripts/strategy_analyzer.py company_data.json

# Generate JSON output for board presentations
python scripts/strategy_analyzer.py company_data.json --output json -f strategy_report.json

# View help and input schema
python scripts/strategy_analyzer.py --help
```

See [references/tools.md](references/tools.md) for detailed tool documentation and input format examples.

### Financial Scenario Modeling

```bash
# Model multiple financial scenarios with NPV, IRR, and risk analysis
python scripts/financial_scenario_analyzer.py scenarios.json

# Generate JSON output for executive dashboards
python scripts/financial_scenario_analyzer.py scenarios.json --output json -f analysis.json
```

## Success Metrics

**Strategic Success**
- Vision clarity and organizational buy-in
- Strategy execution on track (>80% initiatives)
- Market position improving
- Innovation pipeline active and funded

**Financial Success**
- Revenue growth targets met or exceeded
- Profitability trajectory improving
- Cash position strong (18+ months runway)
- Valuation increasing

**Organizational Success**
- Culture thriving (engagement >8/10)
- Key talent retained (attrition <10%)
- High potential succession pipeline
- Diversity and inclusion improving

**Stakeholder Success**
- Board confidence high (no surprises)
- Investor satisfaction and support
- Customer NPS strong (>50)
- Employee approval rating (>85%)

## Red Flags

- Missing targets consistently (3+ quarters)
- High executive turnover (>2 departures/year)
- Board relationship strained
- Culture deteriorating (engagement dropping)
- Market share declining
- Cash burn increasing without growth
- Innovation pipeline stalling
- Personal burnout signs (sustained <7/10 energy)
