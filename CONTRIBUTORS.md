## About This Fork

This repository is a Pandora-focused fork of Ali Rezvani's original claude-skills project. We maintain deep respect for the original work and have retained the MIT license to honor Ali's open-source contribution.

**Changes from Upstream:**
- Focused on 4 core domains (engineering, marketing, product, delivery)
- Removed regulatory compliance domains not applicable to Pandora's operations
- Reorganized folder structure (skills under `skills/` directory, documentation under `docs/`)
- Updated workflows and documentation for Pandora-specific use cases
