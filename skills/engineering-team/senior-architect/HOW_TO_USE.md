## What to Provide

When using this skill, provide:

- **Requirements**: Functional and non-functional requirements
- **Context** (optional): Technical constraints, existing architecture
- **Preferences** (optional): Technology preferences, scalability needs

## What You'll Get

This skill will provide:

- **Architecture Design**: System diagrams, component specifications
- **Technical Documentation**: Architecture decision records (ADRs)
- **Implementation Roadmap**: Phased approach with milestones
- **Automated Tools**: 3 Python scripts for data processing and analysis

## Python Tools Available

This skill includes the following Python tools:

- **architecture_diagram_generator.py**: Architecture Diagram Generator
- **dependency_analyzer.py**: Dependency Analyzer
- **project_architect.py**: Project Architect

You can run these tools directly:

```bash
python skills/engineering-team/senior-architect/scripts/architecture_diagram_generator.py --help
```

## Tips for Best Results

1. **Be Specific**: Provide clear, detailed requirements for better results
2. **Provide Context**: Include relevant background information about your project
3. **Iterate**: Start with a focused request, then refine based on initial results
4. **Combine Skills**: This skill works well with other engineering skills for comprehensive solutions
