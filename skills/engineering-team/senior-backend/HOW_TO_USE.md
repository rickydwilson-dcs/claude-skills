## What to Provide

When using this skill, provide:

- **Codebase**: Link to repository or description of application
- **Context** (optional): Project goals, technology stack
- **Focus Areas** (optional): Specific features or problems to address

## What You'll Get

This skill will provide:

- **Analysis**: Comprehensive evaluation of your request
- **Recommendations**: Actionable guidance and best practices
- **Deliverables**: Formatted outputs and documentation
- **Automated Tools**: 4 Python scripts for data processing and analysis

## Python Tools Available

This skill includes the following Python tools:

- **api_load_tester.py**: Api Load Tester
- **api_scaffolder.py**: Api Scaffolder
- **database_migration_tool.py**: Database Migration Tool
- **performance_analyzer.py**: Performance Analyzer - Backend Performance Analysis Tool

You can run these tools directly:

```bash
python skills/engineering-team/senior-backend/scripts/api_load_tester.py --help
```
