## What to Provide

When using this skill, provide:

- **Infrastructure Details**: Current setup, cloud provider
- **Context** (optional): Scale requirements, budget constraints
- **Goals** (optional): What you want to optimize or automate

## What You'll Get

This skill will provide:

- **Infrastructure Design**: CI/CD pipelines, deployment strategies
- **Automation Scripts**: Configuration files, deployment scripts
- **Monitoring Setup**: Observability and alerting recommendations
- **Automated Tools**: 3 Python scripts for data processing and analysis

## Python Tools Available

This skill includes the following Python tools:

- **deployment_manager.py**: Deployment Manager
- **pipeline_generator.py**: Pipeline Generator
- **terraform_scaffolder.py**: Terraform Scaffolder

You can run these tools directly:

```bash
python skills/engineering-team/senior-devops/scripts/deployment_manager.py --help
```

## Tips for Best Results

1. **Be Specific**: Provide clear, detailed requirements for better results
2. **Provide Context**: Include relevant background information about your project
3. **Iterate**: Start with a focused request, then refine based on initial results
4. **Combine Skills**: This skill works well with other engineering skills for comprehensive solutions
