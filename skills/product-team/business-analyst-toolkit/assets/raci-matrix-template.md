## RACI Legend

- **R (Responsible):** Does the work to complete the task. Can be multiple people.
- **A (Accountable):** Ultimately answerable for the task. Only ONE person per task.
- **C (Consulted):** Provides input and expertise. Two-way communication.
- **I (Informed):** Kept up-to-date on progress. One-way communication.

## RACI Validation Checklist

### Quality Checks
- [ ] Each activity has exactly ONE Accountable (A) person
- [ ] Each activity has at least one Responsible (R) person
- [ ] No role is overloaded (check column totals)
- [ ] All critical stakeholders are included
- [ ] Consulted (C) roles are truly needed (avoid consultation overload)
- [ ] Informed (I) roles receive appropriate updates

### Common Issues to Avoid
- [ ] **Too many As:** Only one Accountable per activity
- [ ] **Missing Rs:** Every task needs someone to do the work
- [ ] **Over-consultation:** Too many Cs slows decision-making
- [ ] **Information overload:** Too many Is creates noise
- [ ] **Unclear activities:** Activities are specific and measurable
- [ ] **Role confusion:** Role names are clear and unambiguous
