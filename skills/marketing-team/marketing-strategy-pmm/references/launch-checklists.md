# Product Launch Checklists Reference Guide

## Overview
Comprehensive checklists and frameworks for planning and executing successful product launches across different launch types, company stages, and market contexts.

## Launch Types & Timing

### Soft Launch (Internal/Beta)
**When:** New product pre-GA, major feature beta
**Audience:** Internal team, design partners, beta customers (50-100 users)
**Timeline:** 2-4 weeks
**Goal:** Validate product-market fit, gather feedback, fix bugs

### Public Launch (General Availability)
**When:** Product ready for all customers
**Audience:** Entire market, press, analysts
**Timeline:** 4-8 weeks planning
**Goal:** Generate awareness, drive adoption, establish market presence

### Feature Launch (Incremental)
**When:** New capability for existing product
**Audience:** Current customers + prospects
**Timeline:** 1-3 weeks
**Goal:** Drive feature adoption, reduce churn, upsell

### Relaunch/Rebrand
**When:** Repositioning, major pivotal, merger
**Audience:** Existing customers, market, press
**Timeline:** 8-12 weeks
**Goal:** Reset market perception, re-engage customers

## Comprehensive Launch Checklist

### T-8 Weeks: Launch Planning

**Product Readiness:**
- [ ] Product scope locked (no feature creep)
- [ ] MVP features validated with customers
- [ ] Quality assurance testing completed
- [ ] Performance benchmarks met
- [ ] Security audit passed
- [ ] Product documentation written
- [ ] Launch date confirmed with engineering

**Positioning & Messaging:**
- [ ] Target customer segments identified
- [ ] Positioning statement finalized
- [ ] Messaging framework created
- [ ] Key differentiators documented
- [ ] Competitive positioning mapped
- [ ] Value propositions tested
- [ ] Proof points collected (metrics, testimonials)

**Launch Strategy:**
- [ ] Launch goals defined (awareness, adoption, pipeline)
- [ ] Success metrics identified (KPIs)
- [ ] Launch budget allocated
- [ ] Launch team roles assigned (PMM, PR, demand gen)
- [ ] Launch timing validated (avoid conflicts)
- [ ] Launch type decided (big bang vs phased)

### T-6 Weeks: Asset Creation

**Content Development:**
- [ ] Website pages (product, feature, comparison)
- [ ] Blog posts (announcement, how-to, use cases)
- [ ] Case studies (2-3 customer stories)
- [ ] White papers/eBooks (thought leadership)
- [ ] Product demo video (2-3 minutes)
- [ ] Tutorial videos (feature walkthroughs)
- [ ] Email templates (announcement, nurture)
- [ ] Social media copy (10-15 posts)

**Sales Enablement:**
- [ ] Pitch deck updated
- [ ] One-pager (product overview)
- [ ] Battle cards (competitive positioning)
- [ ] FAQ document (objections, pricing)
- [ ] Demo script
- [ ] Call scripts (cold, warm, hot)
- [ ] ROI calculator
- [ ] Pricing/packaging finalized

**Technical Assets:**
- [ ] Product screenshots (high-res)
- [ ] Architecture diagrams
- [ ] API documentation
- [ ] Integration guides
- [ ] Security documentation (SOC 2, GDPR)
- [ ] System requirements

**Design Assets:**
- [ ] Product logo/icon
- [ ] Feature graphics
- [ ] Social media graphics (LinkedIn, Twitter banners)
- [ ] Email headers
- [ ] Presentation templates
- [ ] Website banners

### T-4 Weeks: Channel Preparation

**PR & Media:**
- [ ] Press release drafted and approved
- [ ] Media list compiled (tier 1, 2, 3 outlets)
- [ ] Analyst briefings scheduled (Gartner, Forrester)
- [ ] Embargo list confirmed
- [ ] Executive interview prep
- [ ] Press kit assembled (release, images, bios)
- [ ] Media tracking setup (Meltwater, Google Alerts)

**Demand Generation:**
- [ ] Landing pages built and tested
- [ ] Paid ad campaigns created (Google, LinkedIn)
- [ ] Email nurture sequences built
- [ ] Webinar scheduled (launch date or +1 week)
- [ ] Retargeting audiences created
- [ ] Lead scoring model updated
- [ ] HubSpot/Marketo workflows configured

**Customer Marketing:**
- [ ] Customer advisory board briefed
- [ ] Beta customer testimonials secured
- [ ] In-app notifications prepared
- [ ] Customer email announcement drafted
- [ ] Customer webinar/training scheduled
- [ ] Customer success team trained
- [ ] Upgrade/migration plan (if applicable)

**Partner Channel:**
- [ ] Partner announcement email drafted
- [ ] Co-marketing assets created
- [ ] Partner training scheduled
- [ ] Partner incentives defined
- [ ] Joint press release (if applicable)

### T-2 Weeks: Team Enablement

**Sales Training:**
- [ ] Launch training session delivered (1-2 hours)
- [ ] Demo walkthrough completed
- [ ] Competitive positioning reviewed
- [ ] Objection handling practiced
- [ ] Sales enablement materials distributed
- [ ] Slack channel for launch questions created
- [ ] Sales incentives/SPIFs communicated

**Customer Success Training:**
- [ ] Product overview session
- [ ] Customer migration playbook reviewed
- [ ] Feature adoption strategies discussed
- [ ] Support docs reviewed
- [ ] Escalation process defined

**Marketing Team Alignment:**
- [ ] Launch timeline communicated
- [ ] Social media calendar shared
- [ ] Content promotion schedule
- [ ] Paid media calendar
- [ ] Metrics dashboard created
- [ ] Daily standup scheduled for launch week

### T-1 Week: Final Prep

**Quality Assurance:**
- [ ] All landing pages tested (desktop, mobile)
- [ ] Forms tested and connected to CRM
- [ ] Email sends tested
- [ ] Ad campaigns reviewed and scheduled
- [ ] Website updates staged
- [ ] Tracking pixels installed (GA4, LinkedIn, Google Ads)
- [ ] Social posts scheduled

**Launch Day Run-of-Show:**
- [ ] Hour-by-hour timeline created
- [ ] Responsible parties assigned
- [ ] Communication protocols defined (Slack channel)
- [ ] Contingency plans for issues
- [ ] Executive approval for public materials

**Monitoring Setup:**
- [ ] Media monitoring tools configured
- [ ] Social listening activated
- [ ] Website analytics dashboards created
- [ ] HubSpot campaign tracking setup
- [ ] Support ticket tags created

### Launch Day (T-0)

**Morning (6-9am):**
- [ ] Press release goes live on newswire
- [ ] Website updates published
- [ ] Blog post published
- [ ] Email to customers sent
- [ ] Social posts go live (Twitter, LinkedIn)
- [ ] Paid campaigns activated

**Mid-Day (9am-2pm):**
- [ ] Press outreach follow-ups
- [ ] Social media engagement (reply to comments)
- [ ] Influencer/partner amplification
- [ ] Monitor website traffic and conversions
- [ ] Sales team armed with leads from early interest

**Afternoon (2-5pm):**
- [ ] Launch webinar hosted
- [ ] Additional social posts
- [ ] Early metrics review (traffic, leads, press mentions)
- [ ] Address any issues or questions
- [ ] Internal team celebration

### Post-Launch (T+1 Week to 4 Weeks)

**Week 1:**
- [ ] Thank you emails to launch supporters
- [ ] Early adopter outreach and feedback collection
- [ ] Media follow-ups for coverage
- [ ] Metrics report to leadership
- [ ] Social media engagement continuation

**Week 2-4:**
- [ ] Case studies from early adopters
- [ ] Feature adoption tracking
- [ ] Paid campaign optimization
- [ ] Content refresh based on feedback
- [ ] Competitive response monitoring
- [ ] Post-launch retrospective with team

**Ongoing:**
- [ ] Monthly metrics review
- [ ] Quarterly launch impact assessment
- [ ] Feature usage analytics
- [ ] Customer feedback synthesis
- [ ] Sales win/loss analysis

## Launch Metrics Framework

### Awareness Metrics
- Media mentions (tier 1, 2, 3 outlets)
- Social media reach and engagement
- Website traffic (overall and landing pages)
- Organic search impressions
- Earned media impressions

**Targets (B2B SaaS):**
- Tier 1 media: 2-5 mentions
- Social reach: 50k-500k impressions
- Website traffic: +20-50% spike
- Organic search: +100-300 keyword rankings

### Engagement Metrics
- Landing page views
- Content downloads
- Demo requests
- Webinar registrations
- Email open/click rates

**Targets:**
- Landing page CVR: 3-10%
- Content downloads: 50-200
- Demo requests: 20-100
- Webinar attendance: 100-500

### Pipeline Metrics
- Marketing Qualified Leads (MQLs)
- Sales Qualified Leads (SQLs)
- Opportunities created
- Pipeline value generated
- Sales cycle impact

**Targets:**
- MQLs: 100-500
- SQLs: 20-100
- Pipeline: $100k-$1M (varies by ACV)
- Sales cycle: -10-20% faster

### Adoption Metrics (Existing Product)
- Feature activation rate
- Daily/Weekly Active Users (DAU/WAU)
- Time to first value
- Feature retention (D7, D30)
- Upgrade/expansion rate

**Targets:**
- Activation: 20-40% of users
- Retention: 30-50% D30
- Expansion: 5-15% of users upgrade

## Launch Types by Company Stage

### Startup (Pre-Product-Market Fit)
**Focus:** Learning, iteration, beta feedback
**Launch approach:** Soft launch, design partners
**Channels:** Direct outreach, communities, beta lists
**Budget:** $5k-20k
**Timeline:** 2-4 weeks

### Growth Stage (Post-PMF, Pre-Scale)
**Focus:** Market validation, initial awareness
**Launch approach:** Public launch, targeted PR
**Channels:** PR, content, paid ads, webinars
**Budget:** $20k-100k
**Timeline:** 4-8 weeks

### Scale Stage (Series B+)
**Focus:** Market dominance, category leadership
**Launch approach:** Multi-channel, big bang
**Channels:** All (PR, events, partnerships, paid, ABM)
**Budget:** $100k-500k+
**Timeline:** 8-12 weeks

## Crisis Management

### Launch Day Issues
**Website down:**
- Backup: Cloud hosting, CDN, status page
- Communication: Social media update, email to early visitors
- Resolution: Engineering team on standby

**Product bugs:**
- Triage: Severity assessment (blocker vs minor)
- Communication: Transparent acknowledgment, ETA for fix
- Rollback plan: Ability to revert if critical

**Negative press/reviews:**
- Monitor: Set up alerts for brand mentions
- Respond: Address concerns thoughtfully and quickly
- Escalate: PR team handles major issues
