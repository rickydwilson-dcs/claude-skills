## What to Provide

When using this skill, provide:

- **Design Problem**: What needs to be designed or improved
- **Context** (optional): User research findings, brand guidelines
- **Constraints** (optional): Technical limitations, timeline

## What You'll Get

This skill will provide:

- **Design Deliverables**: Wireframes, mockups, prototypes
- **User Flows**: Journey maps, interaction patterns
- **Design System**: Components, patterns, guidelines
- **Automated Tools**: 1 Python scripts for data processing and analysis
